# Portfólio Poke Mania Store: Meu Projeto HTML e CSS

Este projeto é o meu portfólio pessoal, onde apliquei minhas habilidades em HTML5 e CSS3 para desenvolver o frontend da minha própria loja temática, a Poke Mania Store. Meu foco foi criar um código limpo e uma experiência visual que refletisse fielmente o design que imaginei.


# Referência de Design (Figma)

Para garantir a máxima fidelidade visual, o projeto foi desenvolvido diretamente a partir do meu protótipo no Figma.

# Critérios de Avaliação:

Este trabalho foi construído para cumprir os requisitos de uma avaliação técnica, por isso, procurei caprichar na estrutura e no layout para garantir a nota máxima:

Estrutura Semântica: O site é totalmente organizado com as tags corretas do HTML5 (<header>, <main>, <footer>), garantindo um código organizado e funcional.

CSS Limpo e Funcional: Todo o visual foi feito com CSS externo (style.css), utilizando Flexbox para o layout e Variáveis CSS para padronizar as cores.

Desafio do Footer Resolvido: Para centralizar os ícones de Contato na base e manter o Dragonite voando, utilizei uma técnica profissional: position: absolute combinado com transform: translateX(-50%).

Conteúdo Completo: O site possui todas as seções necessárias: Sobre (A História da Minha Loja), Projetos (Produtos/Cards) e Contato.


# Tecnologias:

HTML5: Estrutura base.

CSS3: Estilização e layout (Flexbox).

Git & GitHub: Controle de versão.
